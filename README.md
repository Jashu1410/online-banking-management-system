# Online Banking Management System

This is a simple console-based banking application built using Python and SQLite. 
I created this project to understand how real banking systems handle accounts, 
transactions, and user operations using clean backend logic and object-oriented design.

---

## What this project does

- Allows users to create savings or current accounts
- Supports deposits, withdrawals, and money transfers
- Stores all data safely using SQLite database
- Handles errors and invalid inputs properly
- Uses modular Python files for better code organization

---

## Technologies Used

- Python
- SQLite
- Object-Oriented Programming (OOP)
- Git & GitHub
